<h2>Installation (Vanilla+Superwow)</h2>

1. This addon supports both English/中文
> 
2. Download Latest Version
> 
3. Unpack the Zip file
> 
4. Copy "DoTimerSW" into Wow-Directory\Interface\AddOns
> 
5. Restart WoW
>
